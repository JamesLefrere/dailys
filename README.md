# `dailys`

Farm fresh daily standups delivered to your editor

## Installation

Clone and run `yarn`

## Usage

```bash
yarn run standup

# creates: dailys/standups/{DATE}.md
```

```bash
yarn run standup-edit

# creates: dailys/standups/{DATE}.md and opens it in $EDITOR
```
